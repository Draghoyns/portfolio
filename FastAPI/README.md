# FastAPI Experimentations

## Description
A simple project that follows [FastAPI tutorial](https://fastapi.tiangolo.com/tutorial/)

## Installation

Create a virtual environment and install FastAPI:

```bash
pip install fastapi[standard]
```

or

```bash
conda install fastapi
```


## Usage

Run any of the python scripts with the following command:
```bash
fastapi dev my_script.py
```

Interactive docs should be avalaible at http://127.0.0.1:8000/docs (or whatever the terminal says after the command)
