# Kaggle Titanic — Simple Project

A small project of a basic prediction model using the Kaggle Titanic dataset.

## Contents
- main.py : main code for the model
- data/ — place CSV files here (not included in repo)  

## Dataset
Download the dataset from Kaggle:
- Competition page: https://www.kaggle.com/competitions/titanic
- Direct data link: https://www.kaggle.com/competitions/titanic/data

Requirements:
- A Kaggle account is required

Expected files:
- `data/train.csv`
- `data/test.csv`

## What this project shows
- Basic data analysis
- Simple data preprocessing
- Training a baseline classifier (random forest)
- Training XGBoost classifier
- Cross-validation
- Grid search for hyperparameter tuning
- Generating a Kaggle submission file

## Notes
- Do not commit dataset files to version control.
